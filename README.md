# Power BI Data Professional Survey Dashboard

## Overview
Interactive Power BI dashboard analyzing survey data from data professionals worldwide.

## Tools
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Preview
![Dashboard Preview](screenshots/overview.png)

## Key Insights
- Data Scientists have the highest average salaries
- Python is the most popular programming language
- Work-life balance satisfaction is higher than salary satisfaction
- Most respondents find breaking into data moderately difficult

## Project Structure
- `dashboard/` → Power BI report
- `dataset/` → dataset used for analysis
- `screenshots/` → dashboard images
